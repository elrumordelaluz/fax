# Fax.js

Fax lets transform DOM Elements into Objects/JSON and viceversa, to clone or manipulate it then.

### Element to Object
```javascript
fax.toObject(elem);

```

### Element to JSON
```javascript
fax.toJson(elem);

```

### Object to Element
```javascript
fax.toElement(obj, dest);

```

### Clone Element
```javascript
fax.faxie(inputElement, outputElement, [strict]);

```
